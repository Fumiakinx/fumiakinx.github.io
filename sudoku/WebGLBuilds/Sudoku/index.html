🎮 すうどく 操作説明
🧩 すうどくのルール
  • • 空いているセルに1から9までの数字を入れます。
  • • 縦の列・横の列・3x3のブロックのそれぞれに、同じ数字が重複して入らないように配置します。
💻 PC での操作
  • 🖱️ クリック : セルを選択
  • ⌨️ キー入力 [1]-[9] : 選択したセルに数字を入力
  • ⌨️ Backspace / Delete / [C] キー : 数字をクリア
  • ⌨️ 矢印キー : 選択セルを上下左右に移動
📱 スマホ / タブレット での操作
  • 👉 タップ : セルを選択
  • 👉 下部の [1]〜[9] ボタン : 数字を入力
  • 👉 下部の [Clear] / [C] ボタン : 数字をクリア
